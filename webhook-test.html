🔔 Webhook Connection Test

Verify your Stripe webhook is working without waiting for real payments

ℹ️ How This Works

This tool sends a simulated Stripe webhook event to your endpoint. It creates a fake payment event that your webhook handler will process and save to your database. No real Stripe secret key needed.

1. Select Test Type

🎯 Direct Webhook Test

Test your Supabase Edge Function directly with a simulated Stripe event

📊 Stripe Dashboard Test

Get instructions for testing via Stripe Dashboard (requires login)

2. Configure Test

Stripe Dashboard Test Instructions

  1. Go to Stripe Dashboard → Developers → Webhooks
  2. Find your webhook endpoint for DestinyLens
  3. Click "Send test event"
  4. Select charge.succeeded or payment_intent.succeeded
  5. Click "Send test event"
  6. Check your Supabase income_progress_events table for the test entry

This requires your webhook to be configured in Stripe Dashboard with your actual endpoint URL.

3. Verify Results

After running the test, check your Supabase database:

SELECT * FROM income_progress_events
WHERE provider = 'stripe'
ORDER BY received_at DESC
LIMIT 5;
